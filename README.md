# 🌌 About Nova

Nova is a modern authentication and product hub platform designed to manage user accounts and provide access to multiple applications from a single ecosystem.

```
Create Account
      ↓
Access Apps
      ↓
Manage Everything
```

Nova aims to create a **central identity system** for all future products.

---

# ✨ Features

### 🔐 Authentication
- Secure user registration
- Login system
- Password validation
- Email verification
- OTP verification (planned)

### 👤 User Account
- Profile management
- Update name and profile picture
- Security settings

### 📦 Product Hub
- Browse products
- Product descriptions
- Download applications
- Updates and announcements
